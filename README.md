# Qingchuan Li Academic Homepage

This is a static academic homepage for Qingchuan Li and her research group. It does not require a build step; open `index.html` directly to view the site.

## File Structure

- `index.html`: page content
- `styles.css`: visual styling and responsive layout
- `script.js`: mobile navigation and current year
- `assets/qingchuan-li.jpg`: homepage portrait
